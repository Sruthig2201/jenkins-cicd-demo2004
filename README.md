# Jenkins CI Pipeline with Docker

## Project Overview
This project demonstrates an end-to-end Continuous Integration (CI) pipeline using Jenkins, GitHub, and Docker on Ubuntu Linux.

## Tools Used
- Git & GitHub
- Jenkins
- Docker
- Python
- Ubuntu Linux

## Pipeline Flow
1. Code is pushed to GitHub
2. Jenkins pulls the repository
3. Jenkinsfile defines pipeline stages
4. Docker image is built
5. Docker container is executed

## Files
- Jenkinsfile: Defines Jenkins pipeline
- Dockerfile: Builds Docker image
- app.py: Sample Python application

## Output
The pipeline successfully builds and runs a Docker container.
